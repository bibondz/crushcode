## Phase 41: Auto-Compaction + Smart Context

### What
Auto-trigger compaction when context approaches model limits, replacing heuristic-only with AI-powered summarization.

### Files to Create/Modify
- **MODIFY** `src/agent/compaction.zig` — Add auto-trigger logic, model-specific token limits
- **MODIFY** `src/commands/chat.zig` — Wire auto-compaction into the main loop (before each API call), add token usage bar to prompt
- **CREATE** `src/agent/context_budget.zig` — Model-specific context budgets (max_tokens per model), auto-trim thresholds

### Implementation Details

#### context_budget.zig
```zig
pub const ContextBudget = struct {
    max_context_tokens: u64,        // Model's context window
    reserve_output_tokens: u64,     // Space for model response (e.g., 4096)
    compact_threshold: f64,         // Trigger at 80% of (max - reserve)

    pub fn forModel(model_name: []const u8) ContextBudget;
    pub fn availableTokens(self: *const ContextBudget, used: u64) u64;
    pub fn needsCompaction(self: *const ContextBudget, used: u64) bool;
};
```

Model-specific budgets:
- `MiniMax-M2.7*`: 204,800 context
- `glm-5.1`: 128,000 context
- `claude-*`: 200,000 context
- `gpt-4o`: 128,000 context
- `deepseek-chat`: 128,000 context
- Default: 128,000 context

#### Auto-compaction in main loop
In `chat.zig`, before calling `agent_loop.run()`:
1. Estimate current token usage from `messages`
2. Check `ContextBudget.needsCompaction()`
3. If yes: run compaction, replace messages, show status
4. Continue with API call

#### Token usage bar
Show compact token usage in prompt header:
```
── Crushcode minimax/MiniMax-M2.7 session: session-xxx ──
ctx: ████░░░░ 45% (57K/128K) · /help /compact /cost /exit
```

### Implementation Details

#### project_memory.zig
```zig
pub const ProjectMemory = struct {
    allocator: Allocator,
    user_memory: []const u8,       // ~/.crushcode/CLAUDE.md
    project_memory: []const u8,    // ./.crushcode/CLAUDE.md or ./CLAUDE.md
    combined: []const u8,          // Merged context

    pub fn load(allocator: Allocator) !ProjectMemory;
    pub fn injectIntoSystemPrompt(self: *ProjectMemory, base_prompt: []const u8) ![]const u8;
    pub fn reload(self: *ProjectMemory) !void;
    pub fn deinit(self: *ProjectMemory) void;
};
```

#### CLAUDE.md discovery order (last wins):
1. `~/.crushcode/CLAUDE.md` — User global preferences
2. `./CLAUDE.md` — Project root (if exists)
3. `./.crushcode/CLAUDE.md` — Project-specific (overrides ./CLAUDE.md)

#### Injection into system prompt
Prepend to any existing system_prompt:
```
[Project Context]
{CLAUDE.md content}

{existing system_prompt}
```

#### `/memory` slash command
```
/memory           — Show loaded memory files and their sizes
/memory reload    — Re-scan for CLAUDE.md changes
/memory clear     — Clear injected memory for this session
```

### Success Criteria
- [ ] CLAUDE.md loaded from user + project locations
- [ ] Content injected into system prompt before API calls
- [ ] `/memory` command shows status and allows reload

- **MODIFY** `src/ai/client.zig` — Extract usage from API responses and return to caller

### Implementation Details

#### cost_tracker.zig
```zig
pub const ModelPricing = struct {
    input_per_million: f64,    // USD per 1M input tokens
    output_per_million: f64,   // USD per 1M output tokens

    pub fn forModel(model_name: []const u8) ModelPricing;
    pub fn calculateCost(self: *const ModelPricing, input_tokens: u64, output_tokens: u64) f64;
};

pub const CostTracker = struct {
    allocator: Allocator,
    session_input_tokens: u64,
    session_output_tokens: u64,
    session_cost_usd: f64,
    turn_costs: ArrayList(TurnCost),
    budget_limit_usd: f64,     // 0 = no limit

    pub fn init(allocator: Allocator) CostTracker;
    pub fn recordUsage(self: *CostTracker, model: []const u8, usage: Usage) void;
    pub fn formatCost(self: *const CostTracker) []const u8;
    pub fn isOverBudget(self: *const CostTracker) bool;
    pub fn sessionSummary(self: *const CostTracker) CostSummary;
    pub fn deinit(self: *CostTracker) void;
};

pub const TurnCost = struct {
    turn: u32,
    model: []const u8,
    input_tokens: u32,
    output_tokens: u32,
    cost_usd: f64,
};

pub const CostSummary = struct {
    total_turns: u32,
    total_input_tokens: u64,
    total_output_tokens: u64,
    total_cost_usd: f64,
    by_model: []ModelCostBreakdown,
};
```
