Output: Working `shell.zig` with safe command execution. `build.zig` with all new module registrations.

    ```zig
    /// Execute shell command via process.runShellCommandWithTimeout.
    /// Captures stdout and stderr (up to 1MB each).
    /// Strips ANSI escape sequences from stdout.
    /// Applies midpoint truncation if output exceeds MAX_OUTPUT_CHARS.
    pub fn executeShellCommand(command: []const u8, timeout_seconds: ?u32) !ShellResult {
        const allocator = std.heap.page_allocator;

        const result = try process.runShellCommandWithTimeout(allocator, command, timeout_seconds);
        defer {
            allocator.free(result.stdout);
            allocator.free(result.stderr);
        }

        // Strip ANSI escape sequences from stdout
        const clean_stdout = try ansi_strip.stripAnsiEscapes(allocator, result.stdout);
        defer allocator.free(clean_stdout);

        // Apply midpoint truncation if needed
        const truncated = try truncateOutputAlloc(allocator, clean_stdout);
        // truncated is a new allocation or a dupe — caller must free

        // For stderr: strip ANSI too (in case error messages have color)
        const clean_stderr = try ansi_strip.stripAnsiEscapes(allocator, result.stderr);
        defer allocator.free(clean_stderr);

        return ShellResult{
            .exit_code = if (result.timed_out) 124 else result.exit_code,
            .stdout = truncated, // caller must free
            .stderr = try allocator.dupe(u8, clean_stderr), // caller must free
        };
    }
    ```

    NOTE: The ShellResult stdout/stderr are now actual allocated strings. The old code returned empty string literals `""`. Callers (`handleShell`) don't free them currently — but since they're in a CLI handler that exits, this is acceptable. If needed, add `defer allocator.free(result.stdout);` in handleShell.

    4. **Update `handleShell()`** (lines 296-370) to:

       - Keep existing arg parsing for --timeout/-t unchanged

    5. **Remove `waitWithTimeout()` function** (lines 183-268) — no longer needed, replaced by `process.runShellCommandWithTimeout()`.

    6. **Keep unchanged**: `ShellResult` struct, `truncateOutputAlloc()`, `parseRedirections()`, `countLines()`, `executeInteractiveShell()`.
