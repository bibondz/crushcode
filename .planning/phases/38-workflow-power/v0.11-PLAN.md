## Phase Breakdown

### Phase 38: Agent Modes (Build / Plan / Execute)
**Effort:** Medium

**Current State:**
- `PermissionMode` has `.plan` (read-only) and `.auto`, `.bypassPermissions`

**Implementation:**

5. Wire `AgentMode` into `AgentLoop` â€” filter available tools based on mode
6. Add `/mode` slash command in interactive chat: `/mode plan`, `/mode build`, `/mode execute`
