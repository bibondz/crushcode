### Task 2: Implement Token Tracking

**File:** `src/tui/chat_tui_app.zig`

**What:**
After each AI response is applied (in `applyAssistantResponse`, ~line 2450) and after the system prompt is built (`refreshEffectiveSystemPrompt`, ~line 889), estimate total context token usage.

**Approach:**
1. Create a helper function:
   ```zig
   fn estimateContextTokens(self: *Model) u64 {
       var total: u64 = 0;
       // System prompt tokens
       if (self.effective_system_prompt) |prompt| {
           total += compaction_mod.ContextCompactor.estimateTokens(prompt);
       }
       // History tokens (this is what gets sent to AI)
       for (self.history.items) |msg| {
           total += compaction_mod.ContextCompactor.estimateTokens(msg.content);
       }
       return total;
   }
   ```

2. Call this after `refreshEffectiveSystemPrompt()` completes and after each `applyAssistantResponse()`.

3. Store result in `self.context_tokens`.

**Verification:** Log context tokens after each response (temporary debug), confirm values are reasonable.

### Task 3: Implement /compact Command

**File:** `src/tui/chat_tui_app.zig`

**What:**
Replace the stub at line 2157-2158 with actual compaction.

**Changes:**
Replace:
```zig
} else if (std.mem.eql(u8, name, "/compact")) {
    try self.addMessageUnlocked("assistant", "/compact is not yet implemented.");
```

With:
```zig
} else if (std.mem.eql(u8, name, "/compact")) {
    try self.performCompaction();
```

**New function `performCompaction`:**
```zig
fn performCompaction(self: *Model) !void {
    if (self.history.items.len <= self.compactor.recent_window) {
        try self.addMessageUnlocked("assistant", "Context is too small to compact. Need more conversation history.");
        return;
    }

    // Convert history to CompactMessage slice
    var compact_messages = try self.allocator.alloc(
        compaction_mod.CompactMessage, self.history.items.len
    );
    defer self.allocator.free(compact_messages);
    for (self.history.items, 0..) |msg, i| {
        compact_messages[i] = .{
            .role = msg.role,
            .content = msg.content,
            .timestamp = null,
        };
    }

    // Use rolling summary if available
    var result = try self.compactor.compactWithSummary(
        compact_messages,
        self.last_compaction_summary,
    );

    // Update history — replace old messages with summary as system context
    // Keep only recent messages from result
    if (result.messages_summarized > 0) {
        // Truncate history to recent window only
        const remove_count = result.messages_summarized;
        for (self.history.items[0..remove_count]) |msg| {
            self.allocator.free(msg.content);
            self.allocator.free(msg.role);
        }
        // Shift remaining items
        std.mem.copyForwards(
            core.ChatMessage,
            self.history.items,
            self.history.items[remove_count..],
        );
        self.history.shrinkRetainingCapacity(self.history.items.len - remove_count);

        // Store summary for next compaction
        if (self.last_compaction_summary.len > 0) {
            self.allocator.free(self.last_compaction_summary);
        }
        self.last_compaction_summary = try self.allocator.dupe(u8, result.summary);
    }

    // Update token estimate
    self.context_tokens = self.estimateContextTokens();

    // Report to user
    const report = try std.fmt.allocPrint(self.allocator,
        "Context compacted: {d} messages summarized, {d} tokens saved. Current usage: {d}%.",
        .{
            result.messages_summarized,
            result.tokens_saved,
            self.context_tokens * 100 / self.compactor.max_tokens,
        }
    );
    defer self.allocator.free(report);
    try self.addMessageUnlocked("assistant", report);
}
```

**Edge cases:**
- Handle empty history gracefully
- Don't compact if under recent window (10 messages)
- Preserve tool_call messages (they have role "tool" — don't filter them out)

**Verification:** Run TUI, have conversation >10 messages, type `/compact`, verify:
- Messages summarized count reported
- History reduced (fewer items in history)
- No crash, no memory leak

### Task 4: Auto-Compact Trigger

**File:** `src/tui/chat_tui_app.zig`

**What:**
After each AI response completes (in `applyAssistantResponse`), check if compaction needed and trigger it silently.

**Changes:**
At the end of `applyAssistantResponse()` (~line 2471, after `saveSessionSnapshotUnlocked()`):

```zig
// Auto-compact check
self.context_tokens = self.estimateContextTokens();
if (self.compactor.needsCompaction(self.context_tokens)) {
    // Fire-and-forget compaction (non-blocking for user)
    try self.performCompactionAuto();
}
```

**New function `performCompactionAuto`:**
Same logic as `performCompaction` but:
- Silent (no user message about the compaction itself)
- Show a toast notification: "Auto-compacted: {d} tokens saved"
- Use `.compactLight()` instead of `.compact()` for speed (less disruption)

```zig
fn performCompactionAuto(self: *Model) !void {
    if (self.history.items.len <= self.compactor.recent_window) return;

    var compact_messages = try self.allocator.alloc(
        compaction_mod.CompactMessage, self.history.items.len
    );
    defer self.allocator.free(compact_messages);
    for (self.history.items, 0..) |msg, i| {
        compact_messages[i] = .{
            .role = msg.role,
            .content = msg.content,
            .timestamp = null,
        };
    }

    var result = try self.compactor.compactLight(compact_messages);
    defer result.deinit();

    // Truncate history to match compacted messages
    if (result.tokens_saved > 0) {
        // Remove old history items that were compacted
        const remove_count = self.history.items.len - result.messages.len;
        if (remove_count > 0) {
            for (self.history.items[0..remove_count]) |msg| {
                self.allocator.free(msg.content);
                self.allocator.free(msg.role);
            }
            std.mem.copyForwards(core.ChatMessage, self.history.items, self.history.items[remove_count..]);
            self.history.shrinkRetainingCapacity(self.history.items.len - remove_count);
        }
    }

    self.context_tokens = self.estimateContextTokens();

    // Toast notification
    // (If toast system available from Phase 19, use it. Otherwise skip.)
}
```

**Important:** The auto-compact must NOT lose the last user message or assistant response. The `recent_window` (default 10) guarantees this.

**Verification:** In TUI, have long conversation (>30 messages), verify auto-compact fires and toast shows.

## Risks

- **Token estimation accuracy**: char/4 heuristic is rough. Acceptable for v0.7.0, real tokenizer later.
- **Auto-compact during tool loop**: Must only trigger after final response, not during tool call chain. Guard with `request_active` check.
- **History mutation**: Compaction modifies `history.items` array — must hold lock during operation.
