<objective>
Extract UI chrome widgets (header, sidebar, input) from chat_tui_app.zig into dedicated widget files.

Purpose: These are the "frame" widgets — header bar, sidebar panel, input prompt — that surround the message area. SidebarWidget is the most complex extraction because it currently takes `*const Model` and reads 15+ fields. This is resolved by introducing a SidebarContext data struct that Model.draw() populates.

Output: `src/tui/widgets/header.zig`, `src/tui/widgets/sidebar.zig`, `src/tui/widgets/input.zig`
</objective>
